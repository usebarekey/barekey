Add one named input to `vite.config.ts` and point it at a SvelteKit route.

```ts name="vite.config.ts" icon="vite"
import { sveltekit } from "@sveltejs/kit/vite";
import { defineConfig } from "vite";
import { og } from "svelte-build-og/vite";

export default defineConfig({
	plugins: [
		sveltekit(),
		og({
			input: {
				home: "/og/home",
			},
		}),
	],
});
```

The default capture is `1200 × 630` and the default output is PNG. On machines with a writable browser cache, Playwright installs Chromium automatically when the executable is missing. Read-only and serverless builds need an explicit browser provider, described in [Configuration](/docs/build-og/configuration#serverless-chromium).

The plugin generates `src/.svelte-build-og/generated.d.ts`. Add the directory to `.gitignore`:

```text
src/.svelte-build-og
```
